Welcome to the Special Interest Group on Networks (SIGNET)

Areas of focus

The group is active in a wide range of Networks related areas. The research focus of the group spans from modelling a syatem as a complex network to the lower dimensional representation of networks with recent techniques, such as deep learning methods. Overall activities of the group can be divided into the following three categories.

Multiplex Network Analysis

A multiplex network is a special case of multlayer network in which each layer is restricted to have same set of nodes but can have different set of edges. There is an increasingly sophisticated attempt to model the real-world systems as multiplex network which have have yielded valuable insight in the fields of social network analysis, economics, transport, ecology, medicine, biology, commerce, climatology, physics, computational neuroscience etc.

View Projects

Representation Learning

Representation learning is a set of techniques with the goal of efficiently mapping data structures into convenient latent spaces. This type of embeddings has demonstrated to be useful either for dimensionality reduction or for gaining semantic content. Network Representation Learning (NRL) aims at learning distributed vector representation for each vertex in a network. NRL has shown its effectiveness in many tasks such as image classification and text mining.

View Projects

Large & Distributed Graph Mining

Ranging from social networks to targeted advertising, seeing big graphs to capture the structure in data is not an astonioshing thing. Analysiss of large graphs is central to recent advances in machine learning and data mining. Unfortunately, directly applying existing tools to large graph computation tasks can be cumbersome and inefficient. The need for intuitive, scalable tools for graph computation has lead to the exploration of mining of large graphs.
